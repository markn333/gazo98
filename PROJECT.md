| 現在フェーズ | Phase 1: 基盤構築 |

## 概要

現代のフルカラー画像を、PC-9801のエロゲーCG風（4096色中16色、ディザリング表現）に変換するWebアプリケーション。
98時代のドット絵職人が手作業で行っていた減色＋ディザリングを、アルゴリズムで再現する。

## 技術スタック

- フロントエンド: HTML5 / CSS3 / Vanilla JavaScript（フレームワーク不使用）
- 画像処理: Canvas API + Web Worker
- サーバー: 不要（クライアント完結）
- 対応ブラウザ: Chrome / Firefox / Safari / Edge（最新版）

## 主要機能

1. 8:5固定比率クロップ
2. 16色パレット生成（自動/プリセット6種/手動編集）
3. ディザリング4種（市松/Bayer/Floyd-Steinberg/ランダム）
4. ポストエフェクト3種（スキャンライン/CRTにじみ/ドット強調）
5. スワイプ比較プレビュー
6. ファイル出力3形式（PNG/BMP/MAG）
7. 98風UI（美咲フォント・メニューバー・ダイアログ）
