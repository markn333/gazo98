# GAZO98 実行計画

ステータス: **承認済み**
作成日: 2026-03-13

---

## 概要

PC-9801エロゲー風画像コンバーター「画嬢九八」の開発計画。
現代のフルカラー画像を4096色中16色＋ディザリングでPC-98風に変換するクライアント完結型Webアプリ。

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript（フレームワーク不使用）
- Canvas API + Web Worker
- サーバー不要（静的ファイルのみ）

## リスク・注意事項

1. **MAG形式の実装難度**: XOR差分＋フラグベース圧縮は仕様が複雑。Phase 7で最も時間がかかる可能性あり
2. **美咲フォントのライセンス確認**: Webフォントとしての利用可否を事前確認する
3. **Floyd-Steinberg のパフォーマンス**: 大きな画像でWeb Workerでも時間がかかる場合、プログレス表示が重要
4. **ブラウザ間差異**: Canvas描画やBlob生成でブラウザ間の挙動差に注意
5. **インデックスカラーPNG**: Canvas.toBlobはフルカラー出力のため、インデックスカラーPNG生成には別途エンコーダが必要になる可能性あり

## 補足

- 各Phaseは順番に実施（前Phase完了が次Phaseの前提）
- Phase 1〜4が最小機能セット（MVP）。Phase 5〜8は拡張
- 外部ライブラリは原則不使用（Vanilla JS縛り）。ただしフォントファイルは外部リソース
