# GAZO98 ～画嬢九八～ ユーザーマニュアル

現代のフルカラー画像を、PC-9801のエロゲ風16色画像に変換するWebアプリです。

---

## 基本の使い方

### 1. 画像を読み込む

画面中央のエリアに画像ファイル（JPG/PNG/BMP/WebP/GIF）をドラッグ＆ドロップするか、「ファイルを開く」ボタンをクリックして選択します。

### 2. クロップする

画像を読み込むと、8:5の比率（PC-98の画面比率）でクロップ枠が表示されます。

- **枠の移動**: 枠内をドラッグ
- **枠のサイズ変更**: 四隅のハンドルをドラッグ
- **拡大・縮小**: マウスホイール
- **解像度の選択**: クロップバーのドロップダウンから選択
  - 640×400（標準）
  - 640×200（200ライン）
  - 320×200（ローレゾ）
  - 入力維持

「決定」をクリックするとクロップが確定します。

### 3. パレットを選ぶ

PC-98は同時に16色しか表示できません。どの16色を使うかを決めます。

| 方法 | メニュー操作 | 向いている場面 |
|------|-------------|---------------|
| 自動生成 | パレット → 自動生成 | 画像全体の色をバランスよく |
| 自動生成（肌感重視） | パレット → 自動生成（肌感重視） | 人物画像で肌色を優先 |
| プリセット選択 | パレット → プリセット選択 | 手軽に雰囲気を決めたい |
| 手動編集 | パレット → パレット編集 | 細部にこだわりたい |

### 4. 変換する

「変換実行」ボタンをクリックすると、選んだパレットとパラメータで16色に変換されます。

### 5. 保存する

メニュー「ファイル」から3つの形式で保存できます。

| 形式 | 特徴 | 用途 |
|------|------|------|
| PNG | 16色インデックスカラー | Web掲載、汎用 |
| BMP | 4bitインデックスカラー | Windows環境 |
| MAG | MAKI02形式 | PC-98エミュレータ、実機 |

## パレットの選び方

### プリセットパレット

メニュー「パレット」→「プリセット選択」で、あらかじめ調整された色の組み合わせを選べます。

**汎用プリセット**

| プリセット | 特徴 |
|-----------|------|
| 標準 | バランスの良い万能パレット |
| 暖色系 | オレンジ・茶系が充実。夕景・室内向き |
| 寒色系 | 青・紫系が充実。空・水面向き |
| セピア | モノトーンの暖色。レトロ写真風 |
| 高コントラスト | 原色に近い鮮やかな16色 |
| モノクロ | 白黒16階調 |

**エロゲ風プリセット**

90年代PCゲームの色使いを再現したプリセットです。

| プリセット | 特徴 |
|-----------|------|
| エロゲ標準 | 肌色8色＋その他8色。万能 |
| 暖色エロゲ | 暖色寄り。エルフ/シルキーズ風 |
| 寒色エロゲ | 青白い肌色。SF・ホラー・夜景向き |
| 褐色肌 | 日焼け・褐色肌キャラ向き |

### 自動微調整オプション

プリセット選択ダイアログの下部にある「非肌色枠を画像に合わせて自動調整」をONにすると、プリセットの肌色パレットはそのまま固定しつつ、背景や服の色を画像に合わせて自動最適化します。

→ 肌のクオリティを保ちつつ、背景もきれいに仕上がります。

### パレット編集

パレット → パレット編集で、16色を1色ずつRGBスライダーで調整できます。変更はリアルタイムでプレビューに反映されます。

## 肌感改善

人物画像の肌をきれいに表現するための機能です。コントロールパネルの「肌感改善」パネルで設定します。

### モード選択

| モード | 効果 | おすすめ場面 |
|--------|------|-------------|
| OFF | 通常の自動減色 | 風景・背景のみの画像 |
| 肌色優先 | 肌色に多くのパレット枠を割り当て | 手軽に肌を改善したい |
| 領域分離 | 肌色と非肌色で独立にパレットを最適化 | 最高品質を求める |

### 肌色優先モード

肌色のピクセルに重みを付けてパレット生成するので、肌色の階調が細かくなります。近すぎる肌色は自動的に統合され、空いた枠は背景色に回されるため、重みを上げても無駄な枠消費が発生しません。

- **重み**（1.0〜5.0）: 大きいほど肌色に多くのパレット枠が割り当てられます。

| 重み | 効果 |
|------|------|
| 1.0 | 肌色優先なし（通常の自動生成と同じ） |
| 2.0 | 穏やかに肌色を優先。風景と人物が半々の画像向き |
| 3.0（初期値） | 標準。多くの画像でバランスが良い |
| 4.0 | 肌色の階調を最大限確保。ポートレート向き |
| 5.0 | 肌色最優先。アップの顔画像など肌が主役の場面に |

> **仕組み**: 重みを上げると、肌色ピクセルがパレット生成時に優先されます。ただし、近すぎるパレット色（RGB距離20以下）は自動的に1色に統合され、空いた枠は背景色のメディアンカットで再配分されます。そのため、重み5.0でも「ほぼ同じ肌色が何色も並ぶ」ことなく、背景の品質も保たれます。

- **最低枠数**（0〜8色）: 肌色に最低限割り当てるパレット数を保証します。4色が標準。

### 領域分離モード

画像を「肌色の部分」と「それ以外」に分離し、それぞれ独立にパレットを最適化します。肌色優先モードより高品質ですが、処理がやや重くなります。

- **配分: 自動**: 肌色ピクセルの割合から自動計算（1.5倍の補正付き）
- **配分: 手動**: 「肌枠」スライダーで肌色に割り当てる色数（4〜12色）を直接指定

### 肌専用ディザ

肌色の部分だけに専用のディザリングパターンを適用します。通常のディザとは別に、肌に最適化されたパターンで色の段差を目立たなくします。

| パターン | 見た目 |
|---------|--------|
| スムース | 規則的だが目立ちにくいパターン。最も自然 |
| ハーフトーン | 印刷物風の丸いドット配置。柔らかい印象 |
| 斜め45° | 斜め方向にドットが整列。水平走査線と干渉しにくい |

- **ブレンド**（2〜8px）: 肌色領域と非肌色領域の境界をどれだけ滑らかに繋ぐかを指定します。小さいと境界がはっきり、大きいと自然に溶け込みます。

### マスク確認

「マスク確認」ボタンを押すと、肌色と判定された部分が赤いオーバーレイで表示されます。肌色検出がうまくいっていない場合は「検出設定」で調整できます。

### 検出設定（上級者向け）

肌色の判定条件をHSV色空間のパラメータで調整します。

| パラメータ | 初期値 | 効果 |
|-----------|--------|------|
| H（色相）min/max | 0°〜50° | 検出する色相の範囲。広げると黄色・茶色も肌色として検出 |
| S（彩度）min/max | 0.10〜0.70 | 彩度の範囲。白っぽい肌や濃い肌を含めるか |
| V（明度）min/max | 0.20〜0.95 | 明度の範囲。暗い影や明るいハイライトを含めるか |

## ディザリング

16色に減色すると色の段差（バンディング）が目立ちます。ディザリングは2色のドットを交互に配置して、中間色を擬似的に表現する手法です。

### ディザリング方式

| 方式 | 特徴 | 98エロゲでの使用例 |
|------|------|-------------------|
| 市松パターン | 座標の偶奇で2色を切り替え。最もPC-98エロゲらしい | 大半の16色CG |
| Bayer行列 | 規則的なマトリクスで閾値判定。やや機械的な見た目 | ツール自動生成画像 |
| Floyd-Steinberg | 誤差拡散。最も自然だがPC-98らしさは薄い | 高品質変換ツール |
| ランダム | ランダムにノイズを加える。独特の粒状感 | 特殊効果 |

### 強度

0%〜100%で調整します。

- **0%**: ディザなし（ベタ塗り）。色の段差がくっきり
- **50%**: 適度なディザ。段差が和らぐ
- **80%**（初期値）: しっかりディザ。グラデーションが滑らか
- **100%**: 最大。ドットパターンが目立つ

## ポストエフェクト

変換後の画像にCRTモニター風の効果を加えます。3種のエフェクトは独立にON/OFFでき、組み合わせられます。

| エフェクト | 効果 |
|-----------|------|
| スキャンライン | 1行おきに暗い線を入れ、CRTの走査線を再現 |
| CRTにじみ | 赤・青チャンネルを左右にずらし、にじんだ発色を再現。さらにぼかし合成でグロー効果 |
| ドット感強調 | シャープネスフィルタでドットの境界を強調 |

**おすすめの組み合わせ:**

- スキャンライン のみ → シンプルなCRT感
- スキャンライン ＋ CRTにじみ → 本格的なCRT再現
- 全部ON → 雰囲気全開（重くなる場合あり）

> **注意**: BMP/MAGで保存する場合はエフェクトなしの16色画像が保存されます。PNGで保存する場合は、エフェクトON中にダイアログで「エフェクト付き（フルカラー）」または「エフェクトなし（16色インデックス）」を選べます。

## プレビュー＆比較

変換後にメニュー「表示」から利用できます。

| 機能 | 操作 |
|------|------|
| 原画像表示 | 表示 → 原画像 |
| 変換後表示 | 表示 → 変換後 |
| スワイプ比較 | 表示 → スワイプ比較。分割線をドラッグして左右を比較 |
| ズーム | 表示 → 拡大表示（1x→2x→4x→8x→1xでサイクル）。マウスホイールでも操作可能 |
| パン | ズーム中にドラッグで表示位置を移動 |

**スマホでの操作:**

| 操作 | 比較モード | その他モード |
|------|-----------|-------------|
| 1本指スワイプ | 分割線を左右に移動 | 画像をパン移動 |
| 2本指ドラッグ | 画像をパン移動 | 画像をパン移動 |
| ピンチイン/アウト | ズーム | ズーム |

> **スマホ向け**: 画面幅768px以下ではメニューがハンバーガーメニュー（☰）に切り替わります。各パネル（パレット・肌感改善・パラメータ）はタイトルをタップすると折り畳みできます。

## おすすめの変換手順

### 初心者向け：手軽にエロゲ風

1. 画像を読み込み → クロップ
2. パレット → プリセット選択 → 「エロゲ標準」→ OK
3. 変換実行
4. 好みならファイル → 保存(PNG)

### 中級者向け：肌をきれいに

1. 画像を読み込み → クロップ
2. 肌感改善モード「肌色優先」に設定
3. ディザを「市松パターン」、強度80%に
4. 肌専用ディザをON（パターン: スムース）
5. 変換実行
6. スワイプ比較で確認 → 必要なら重みや最低枠を調整して再変換

### 上級者向け：最高品質

1. 画像を読み込み → クロップ
2. 肌感改善モード「領域分離」、配分: 手動、肌枠: 8色
3. ディザを「Floyd-Steinberg」、強度80%
4. 肌専用ディザON（パターン: スムース、ブレンド: 4px）
5. 変換実行
6. マスク確認で肌色検出を確認 → 必要なら検出設定を調整
7. 表示 → スワイプ比較 → ズームでドット単位で確認
8. 満足したらMAG形式で保存 → 98エミュレータで表示確認
